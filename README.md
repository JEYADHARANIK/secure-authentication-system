# Secure User Authentication System

This project is a backend authentication system built using Node.js, Express, and MongoDB.
## Features
- User Registration
- User Login
- Password Hashing
- JWT Authentication
- Forgot Password
- Role-based Access (User/Admin)

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT
